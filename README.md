# Pilot-Training-Simulator
A custom pilot training simulator built with Python, C, and PyQt — reducing aviation training costs by replicating real-world flight routes and live data for a methane detection company.


## Overview
Training pilots in real aircraft is expensive. This simulator replicates 
the company's actual flight routes and missions, allowing new pilots to 
practice maintaining correct elevations, approach angles, and flight 
techniques specific to low-altitude detection operations — without fuel 
or aircraft expenses.

The custom PyQt interface displays real-time flight data including live 
route tracing and elevation metrics, and is intentionally designed to 
mirror the look and feel of the company's existing software — so the 
transition from simulator to real operations is as seamless as possible.

## Features
- Real-time navigation map with live flight data and route tracing
- Replicates real-world methane detection flight paths and elevations
- Custom UI modeled after the company's actual software for intuitive onboarding
- Built on Microsoft Flight Simulator SDK for a realistic flight experience

## Tech Stack

- PyQt
- Microsoft Flight Simulator SDK
